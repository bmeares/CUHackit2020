# Hackbox Games
Hackbox games is a ~~Jackbox Games clone~~ hacker-themed
multiplayer game platform. 

Current implementation has two games, a simple trivia game
and a hacking-themed spyfall clone. 

## Tools Used
* Docker
* MySQL/SQLAlchemy
* Flask
* HTML/CSS/JS
* OpenTriviaDB API
